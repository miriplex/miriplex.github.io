Connected Status Reporter

Year: 2024

Description: A Telegram bot developed to track server updates for the Orb project, delivering real-time notifications to testers. The bot collects and analyzes data, provides location updates on OpenStreetMap, and collects user feedback for training AI models. It was developed using Python and hosted on Google Cloud Run.

Tech Stack: Python, Google Cloud Run
