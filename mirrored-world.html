Mirrored World

Year: 2020

Description: Mirrored World is a puzzle game created for a weekly game jam on itch.io. The game features a mirrored environment where players solve intricate puzzles by navigating mirrored pathways. The game was highly praised by the community, earning the second-highest number of downloads during the game jam, and received overwhelmingly positive reviews.

Platform: Windows

Tech Stack: Unity (C#)
