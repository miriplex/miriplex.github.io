Orb

Year: 2021 - Present

Description: Orb is a mobile app that displays live status updates of users directly on a homescreen widget. By analyzing user data such as geolocation, activity, and music preferences (with the user's permission), the app determines what the user is currently doing and shares this status with their friends. I was responsible for the backend development, including managing data storage, user authentication, and backend logic using Firebase and Google Cloud. The frontend was developed by other team members using Flutter. The app is currently in Beta testing.

Platform: Mobile (iOS/Android)

Tech Stack (Backend): Firebase (for data storage and authentication), Google Cloud (for backend services)
